Package.json filen er I centrum af et hvert Node.js projekt, den indenholder vigtig metadata omkring projektet der er nødvendigt. Den definerer også attributter af et projekt som nem bruger til at installere de nødvendige dependencies og køre scripts. Fetch bruges til at requeste data fra en server, requesten kan være alle slags API’er sålænge den returnerer dataen i enten JSON eller XML. Fetch kræver et parameter, og det er den url den skal requeste, den returnere et promise. CSS kan bruges på tre forskellige måder:
1. Inline CSS - som bruges til at style et specifikt element.
2. Style tag i HTML filen - som bruges til at style en specifik html side.
3. Stylesheet/seperat CSS fil/ekstern CSS - kan bruges til at style flere html sider, potentielt et helt projekt.
Som udgangspunkt vil et stylesheet være at foretrække da man med brug af id og klasser i ens html filer, kan style helt ned på element niveau uden det store ekstra arbejde.
